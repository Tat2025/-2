Git

  • GIT LS-FILES – расшифровывается как "List files", используется для отображения списка файлов в текущем рабочем каталоге Git.
  • GIT MV – расшифровывается как "Move or rename files". Эта команда используется для перемещения или переименования файлов в репозитории Git.
  • GIT INIT – cоздает новый репозиторий Git или преобразует существующий каталог в репозиторий Git.
  • GIT CLONE – клонирует репозиторий Git с удаленного сервера на локальную машину.
  • GIT ADD – добавляет файлы в индекс Git перед фиксацией изменений.
  • GIT COMMIT – фиксирует изменения в индексе Git и сохраняет их в истории репозитория.
