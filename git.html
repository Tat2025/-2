Git

  • git status– Показывает состояния файлов в рабочем каталоге и индексе.

  • git mv– Удобный способ переместить файл в репозитории Git.

  • git init – Создает новый репозиторий Git или преобразует существующий каталог в репозиторий Git.

  • git clone – конфигурации репозиторий Git с удаленного сервера на локальную машину.

  • git add – Добавляет файлы в индекс Git перед фиксацией изменений.

  • git commit – Фиксирует изменения в индексе Git и сохраняет их в истории репозитория.

  • git clean – Удаляет мусор из рабочего каталога.

  • git rm – Удаляет файлы из индекса и рабочей копии.

  • git push – Используется для установления связи с удалённым репозиторием, вычисления локальных изменений отсутствующих в нём, и их передачи.

  • git branch – Перечисляет ветки, создает новые, удаляет и переименовывает их.

  • git fetch – Связывается с удалённым репозиторием и забирает из него все изменения, которых пока нет и сохраняет их локально.
